Earn coins, keys and more for free in Subway Surf!

How to Use

  1. Download Subway Surfers.
  2. Open your file manager app.
  3. Navigate to: Android > data > com.kiloo.subwaysurf > files > profile
  4. Download or copy the generated JSON code.
  5. Paste the file in the "profile" folder.
